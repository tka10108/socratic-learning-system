# 苏格拉底学习系统

> 一个让Agent化身为你的私人老师，用苏格拉底式提问引导你主动思考的学习框架。

---

## 这是什么？

这不是一个 App，也不是一个插件——它是一套**给 Agent 看的文件系统**。

当你把这些文件放在项目目录下，用 任何支持本地文件读取的 Agent（Claude Code、Opencode、Codex 等）开始对话时，它会读取这些文件，理解你的老师设定、学习进度和学习风格，然后以苏格拉底式教学法引导你学习任何你想学的内容。

**核心理念：不直接给你答案，而是通过一系列问题引导你自己想出来。**
你可以让Agent直接按照这个README文档进行配置。

---

## 快速开始

### 第一步：克隆项目

```bash
git clone https://github.com/your-username/socratic-learning-system.git
cd socratic-learning-system
```

### 第二步：配置你的信息

编辑 `USER.md`，填入你的基本信息：
- 你是谁（年龄、学校、背景）
- 你的学习目标
- 你的作息时间（可选，帮助老师规划进度）

### 第三步：创建你的老师

在 `teacher/` 文件夹下，参考 `teacher/TEMPLATE.md` 创建你自己的老师档案。

你可以创建任意数量的老师，每位老师有独立的性格、说话风格和专业背景。

### 第四步：创建课程

在 `courses/` 下新建一个文件夹（课程名），包含三个文件：
- `progress.md`：学习进度
- `reference.md`：参考资料和笔记
- `diary.md`：课后日记

参考 `courses/example-course/` 里的模板。

### 第五步：开始上课

用 Agent 客户端打开这个目录，然后对 Agent 说：

```
读取 system.md，我想开始上课
```

Agent 会读取所有配置文件，询问你选择哪位老师，然后开始教学。

## 核心功能

### 苏格拉底式教学
老师不直接给答案，通过逐层递进的问题引导你自己推导出结论。你主动思考得到的答案，比被动接受的讲解记得更牢。

### 科学学习方法
系统内置四种经过研究验证的学习方法：
- **间隔练习**：每节课回顾之前的内容，防止遗忘
- **主动回忆**：让你先说出概念，再对照确认
- **交错练习**：混合不同类型的问题，避免机械记忆
- **自我解释**：不只是给答案，还要说出"为什么"

### 学习进度追踪
每节课结束后，Agent 会自动更新 `progress.md` 和 `reference.md`，记录你学到的内容和下次继续的位置。

### 课后日记
Agent 会以你的视角写课后日记（`diary.md`），记录学习过程中的困惑、顿悟和感受，帮助你回顾和反思。

### 模拟微信群（可选）
老师可以在模拟的"微信群"里发消息，营造更真实的师生关系。

## 教师设定建议

老师是这个系统最有趣的部分。你可以把老师设定成：
- 你喜欢的某位历史人物
- 一个有特定背景的虚拟角色（如"北大物理系博士生"）
- 严厉型、温柔型、幽默型……任何你喜欢的风格

**好的老师设定应该包含：**
1. 基本背景（学校、专业、年龄）
2. 性格特点（说话方式、口头禅）
3. 动作描写风格（增加沉浸感）
4. 专业擅长领域

详见 `teacher/TEMPLATE.md`。

## 使用提示

- **一次只上一门课**：Agent 会专注于当前课程的进度
- **教材放在课程文件夹里**：Agent 支持读取 PDF，把教材放在 `courses/课程名/` 下
- **说"先到这里"结束课程**：Agent 会更新进度文件再结束
- **说"查看学习进度"**：随时查看当前课程进度
- **说"我想看看微信群聊"**：触发群聊功能

## 适合什么场景？

- 自学某门课程（数学、编程、语言、物理……）
- 读一本书，想有人帮你深化理解
- 备考，需要系统性复习
- 任何"你想真正搞懂，而不只是看一遍"的知识
